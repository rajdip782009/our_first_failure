# Educational File Sharing Bot

This is a Telegram bot for educational purposes to share files via links with force sub and verification.

## Deployment
See steps in the guide.

## Variables
- TG_BOT_TOKEN: Your bot token
- APP_ID: API ID
- API_HASH: API Hash
- CHANNEL_ID: Database channel ID
- LOG_CHANNEL_ID: Log channel ID
- FORCE_SUB_CHANNEL: Update channel ID
- DATABASE_URL: MongoDB URL
- DATABASE_NAME: DB name
- OWNER_ID: Your user ID
- ADMINS: Space-separated admin IDs
- PROTECT_CONTENT: True/False
- AUTO_DELETE_TIME: Seconds for auto delete
